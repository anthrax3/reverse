
# reverse

  Unicode-aware string reverser. For more information, visit: https://github.com/mathiasbynens/esrever.

## Installation

  Install with [component(1)](http://component.io):

    $ component install component/reverse

## Example

```js
reverse('foo 𝌆 bar') //
reverse('mañana mañana') //
```

## API

### `reverse(str)`

Reverse a string
